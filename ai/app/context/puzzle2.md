# Puzzle 2 — Cosmic Clue: The Deep Space Incident

## In-World Framing
Case File X-901. A target was neutralized on Station Delta-9. The incident log
is classified. The crew must deduce the exact combination of suspect, station
module, and implementation (weapon) using the clues embedded in the dossier.

## Elements

### Suspects
Dr. Green, Colonel Mustard, Mrs. Peacock, Professor Plum, Agent Scarlet

### Station Modules
Cryo-Stasis Bay, Observation Deck, Holographic Rec-Module, Astro-Lounge,
Biodome

### Implementations (Weapons)
Sonic Pistol, Cryo-Saber, Kinetic Pipe, Fusion Coil, Gamma Lash

### Casualties (Victims)
Buddy, Cook, Dr. Orchid, Eyewitness, Fiancé

## Mechanics
- This is a logic grid puzzle. Players eliminate combinations using clues.
- Clues are embedded in the dossier text and visual notes around the grid.
- The solution is one unique combination: suspect + module + implementation.
- Players mark the grid (checkmarks for confirmed, X for eliminated).
- The puzzle is solved when only one valid combination remains.

## Oracle Hint Guidance

### Tier 1 (vague, thematic)
- Reference the classified dossier and the need to read between the lines.
- Do not name any suspect, module, or weapon.
- Example: "The incident log reveals its truth only to those who read
  not what is written, but what is absent, Seeker."

### Tier 2 (slightly more concrete)
- Ask which suspects or modules the crew has already eliminated.
- Suggest focusing on the alphabetical clue or the navigation control clue
  without stating the answer.
- Example: "Consider the order of names as the Beacon's logs record them.
  Alphabetical precedence has guided navigators through many a dark passage."

### Tier 3 (mechanic guidance, no full answer)
- Remind players that each clue eliminates at least one combination.
- Suggest they re-read the clue about who was found near Navigation Control
  and what that rules out.
- Example: "The one found near Navigation Control carries an alibi written
  in their very location. Cross-reference this with what the dossier says
  of Professor Plum's associate."
